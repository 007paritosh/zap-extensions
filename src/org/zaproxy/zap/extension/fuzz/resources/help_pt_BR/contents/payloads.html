Diálogo de cargas/payloads

É o que permite a você selecionar os geradores de carga para usar ao fazer fuzzing num pedido/request.

Geradores de carga/payload geram os ataques que o difusor envia para o aplicativo de destino.

Os seguintes tipos de geradores são fornecidos por padrão:
  • Arquivo - selecione qualquer arquivo local para um ataque
  • Arquivos de Fuzzers - selecione qualquer combinação dos arquivos de difusão registrados com o ZAP, por exemplo, através de complementos como o fuzzdb
  • Regex - gerar ataques baseados em padrões regex
  • Strings - são strings que podem ser digitadas ou coladas
  • Script - scripts personalizados que pode gerar quaisquer payloads/cargas
Você pode escrever scripts geradores de carga personalizados - eles podem proporcionar quaisquer cargas que você precisar.

Complementos também podem definir geradores de carga adicionais.

O botão 'Processadores...' lança o diálogo de processadores de carga que permite a você configurar processadores de carga que só servem para o gerador de palyload que você selecionou.
