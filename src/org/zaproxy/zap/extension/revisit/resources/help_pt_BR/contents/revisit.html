Revisitar

Você alguma vez quis navegar um aplicativo que você testou com o ZAP em um momento específico - talvez quando você não tenha mais acesso a ele, ou depois que ele foi alterado de alguma maneira?
-Então este complemento é para você!

O Add-on permite a você instruir o ZAP para retornar o conteúdo do histórico entre datas especificadas, ao invés de encaminhar os requests para o site.
Isso permite que você aparentemente navegue e use um site ao qual já não tem mais acesso.
Também permite que veja como ele era entre momentos determinados, o que é útil se o aplicativo mudou desde então.

Ao retornar páginas do histórico do ZAP, rodapés flutuantes são inseridos nas páginas HTML, para dar uma indicação visual que estas páginas são geradas pelo ZAP.
O ZAP retornará páginas que foram o resultado de solicitações de GET e de POST sempre que encontrar aqueles que correspondam a essas solicitações.
A correspondência acontecerá apenas se todos os parâmetros de URL e POST forem os mesmos, ignorando a ordem e quaisquer tokens CSRF conhecidos.
Se nenhuma página correspondente puder ser encontrada no histórico, uma página 404 personalizada será retornada, com os links para a mesma página, caso tenha sido chamada com parâmetros diferentes ou fora do período de tempo especificado.

Este complemento adiciona menus de botão direito na Árvore de Sites:

Habilitar revisita

When selected a popup will be shown which asks for the start and end times to be used.
The start time defaults to the time of the first request in the session.
The end time defaults to the time of the last request in the session.
If any of the time elements (for example 'day of the month') have not changed then those elements will be disabled.
If the dialog OK button is pressed then the revisit functionality will be enabled for the selected site, and a 'calendar' icon added to the site node in the History tree.

Desativar revisita

When selected then the revisit functionality will be disabled for the selected site.

API

O complemento na suporta os seguintes endpoints de API:

revisit / view / revisitList Retorna a lista de sites habilitados para revisita
revisitar / ação / revisitSiteOn Habilita revisita para o site especificado. Datas devem usar o formato: "AAAA/MM/dd hh:mm:ss"
revisitar / ação / revisitSiteOff Desabilita revisita para o site especificado
