Aba do Console de Script

O Console de Scripts permite que você escreva scripts que são executados dentro do ZAP.
Ele é composto de:

  • Uma barra de ferramentas
  • Uma área em que você pode escrever seus scripts
  • Uma área para depurar e mensagens de erro
Você pode executar scripts 'stand alone' de dentro desta guia, usando o botão 'Run' na barra de ferramentas da aba.
Todos os outros tipos de scripts serão executados quando habilitados ou se invocados explicitamente.

Para criar, carregar ou trocar de script, veja a Aba da Árvore de Scripts.

Templates podem ser vistos no console, mas não podem ser editados.
